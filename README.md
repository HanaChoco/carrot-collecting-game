# Carrot Collector Game

オープンカーでにんじんを集めるゲーム

**Version:** 1.0.0  
**Creation Date:** 2023年1月  
**Programming Language:** Processing  

学部の講義の自由課題で作成

## Overview

### Rules
- **操作方法:**
  - マウスカーソルで車を移動させ、車道に落ちているにんじんを集める  
- **クリア条件:**
  - 20個のにんじんを集める  
- **体力ゲージ:**
  - にんじんを取り忘れると体力が減少  
  - 体力が0になるとゲームオーバー
